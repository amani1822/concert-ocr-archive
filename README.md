# OCR 기반 콘서트 기록 자동 정리 애플리케이션

## 1. 프로젝트 개요
공연 티켓 이미지나 예매 화면을 업로드하면 OCR을 통해 텍스트를 추출하고, 공연 날짜·아티스트·공연명·장소·티켓 가격을 자동으로 입력하는 콘서트 기록 정리 애플리케이션입니다.

## 2. 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

Tesseract OCR 프로그램이 설치되어 있지 않은 환경에서는 샘플 텍스트로 시연할 수 있도록 예외 처리를 추가했습니다.

## 3. 주요 기능
- 티켓 이미지 업로드
- OCR 텍스트 추출
- 공연 정보 자동 입력
- 사용자 직접 수정
- 공연 기록 저장
- 아티스트별 관람 횟수 순위
- 월별/누적 공연 소비 금액 통계

## 4. 사용 기술
- Python
- Streamlit
- pytesseract
- pandas
- CSV 파일 저장
